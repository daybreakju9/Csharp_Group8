# 图像筛选系统 - 桌面客户端使用说明 (DesktopClient)

## 1. 简介

本客户端是图像筛选系统的桌面端操作界面，主要供管理员和用户使用，提供图片浏览、筛选、分类和管理功能。

## 2. 安装与运行

### 系统要求
- Windows10/11
- .NET8.0

### 启动
1. 运行目录下的 `ImageAnnotationApp.exe`。
2. 首次运行可能需要配置服务器地址（如果未在配置文件中指定）。

## 3. 功能模块说明

### 3.1 用户登录/注册
- **登录**: 输入用户名和密码，登录成功后会自动保存会话。
- **注册**: 新用户可以通过注册界面创建账号，是游客身份，需要管理员审核后获得用户身份。

### 3.2 项目管理 (Project Management)
- **创建项目**: 管理员可以创建新的筛选项目，设定项目名称和描述。
- **导入图片**: 支持批量导入本地图片到指定项目。
  - 图片会自动上传至服务器。
  - 支持 `jpg`, `png`, `bmp`, `webp` 等常见格式。

### 3.3 队列管理 (Queue Management)
- **创建队列**: 将项目中的图片分配到不同的工作队列中。
- **分配任务**: 可以将队列分配给特定的用户进行处理。
- **优先级**: 设置队列的处理优先级。

### 3.4 图片筛选与标注 (核心功能)
- **浏览模式**:
  - 支持键盘快捷键快速切换图片 (← / →)。
  - 支持缩放查看细节 (鼠标滚轮)。
- **筛选操作**:
  - **保留/通过**: 标记图片为合格。
  - **淘汰/拒绝**: 标记图片为不合格。
  - **分组**: 支持将相似图片归类到同一组。
- **进度保存**: 操作实时同步到服务器，支持断点续传。

### 3.5 数据导出
- 支持将筛选结果导出为 Excel/CSV 报表。
- 支持按筛选结果（如“仅合格图片”）批量下载/导出文件到本地文件夹。

## 4. 常见问题 (FAQ)

**Q: 无法连接服务器？**
A: 请检查网络连接，并确认后端服务已启动且地址配置正确。

**Q: 图片加载缓慢？**
A: 客户端启用了本地缓存机制。首次加载可能较慢，再次查看时会从本地缓存读取。请检查服务器带宽配置。

**Q: 如何更新客户端？**
A: 直接替换新的 `.exe` 和相关 `.dll` 文件即可，无需重新安装。
